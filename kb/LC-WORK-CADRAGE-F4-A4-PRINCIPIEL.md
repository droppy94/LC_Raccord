## 3. Cibles primaires, issues pré-déclarées, critère de verdict `[FIGÉS]`

### F4-G1 — Non-circularité du principe
**Question.** Le principe d'entropie fournit-il une **contrainte indépendante** qui sélectionne
`g₃→0`, ou **re-exprime-t-il** A4 (puisque `S_grav` est bâtie sur le Weyl) ?
- **G1-a = réduction.** Un principe **indépendant** (non bâti sur le Weyl) force le faible-Weyl —
  **dérivation non-circulaire**.
- **G1-b = délimitation.** Le principe **motive / justifie mieux** A4 sans le dériver ; A4 reste un
  **socle, mieux argumenté** (*issue honnête attendue*).
- **G1-c = négatif / circularité.** `S_grav` définie via le Weyl ⟹ « `S_grav` petit ⟺ Weyl petit »
  est une **tautologie** ; **aucun gain principiel**.

### F4-G2 — Déplacement vs suppression du postulat (*Past Hypothesis*)
**Question.** Le 2ⁿᵈ principe + `S_grav` **croissante** *requiert* une **basse entropie initiale** =
exactement A4. Le principe **déplace-t-il** le postulat (de « faible Weyl » vers « basse entropie
initiale ») **sans le supprimer** ?
- **G2-a = réduction.** Un **mécanisme** (p.ex. structure inter-éons CCC ; ou un attracteur dynamique
  back-réaction→`N`) **dérive** la basse entropie initiale comme **conséquence**, pas postulat.
- **G2-b = délimitation.** La basse entropie initiale est **mieux située** (reliée à (a3) `P=P*` ou à
  la stabilité #5) mais **reste posée**.
- **G2-c = négatif / déplacement.** Le principe **renomme** A4 en *Past Hypothesis* ; l'**asymétrie
  thermodynamique du temps** reste un **intrant** non dérivé.

### Critère de verdict `[GELÉ — tripartite]`
Pour le principe d'entropie examiné :
1. **(i)** sélectionne `g₃→0` **sans** intrant équivalent à A4 (principe **indépendant** du Weyl, basse
   valeur initiale **non posée**) ⟹ **dérivation** (G1-a / G2-a) ;
2. **(ii)** **présuppose** une basse entropie / faible Weyl initial (*Past Hypothesis*) ⟹
   **déplacement / délimitation** (G1-b / G2-b ou G2-c) ;
3. **(iii)** `S_grav` **définie via le Weyl** ⟹ « `S_grav` petit ⟺ Weyl petit » **tautologique** ⟹
   **circularité** (G1-c).

**Verdict composé** = lecture conjointe (G1 ; G2). Réduction candidate **seulement** si `(i)` sur G1
**et** G2-a (dérivation **et** condition initiale dérivée). Tout le reste = **délimitation** ou
**négatif/circularité/déplacement**, **sans** réduction de compte.
