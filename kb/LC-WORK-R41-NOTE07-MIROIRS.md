## §2 — Corrections FACTUELLES à injecter au paquet CSE (sans changement de scope)

**C-1 — ATTRIBUTION (S2).** Le cadrage §6.1 et NOTE-07 attribuent `2504.07630` à « Chen–Penington ».
**Faux** : l'auteur est **Antony J. Speranza, seul**. Speranza **s'appuie** sur le modèle de

⟹ pas d'amendement R-7.

**C-2 — FAITHFULNESS (S1), load-bearing.** Le cadrage libelle `2512.04435` : « mode ajouté = mode
gravitationnel OU observateur, interchangeables ». Le résumé **ne pose pas cela**. Il pose : algèbres
gravitationnelles sur espace-temps à **deux surfaces extrémales** ; exemple = **long wormhole à deux
bords AdS asymptotiques** ; propriété de scission, algèbres de type II par produit croisé (extérieur
gauche, extérieur droit, région médiane dite « python's lunch », compléments) ; cas où seul
l'opérateur **somme d'aires** ou **différence d'aires** est retenu (ensemble microcanonique), l'algèbre
étant alors type II ou type III selon la région ; le mode différence d'aires donne un poids se
restreignant à une **trace** sur la région médiane ; les différences d'entropies relatives donnent des
différences d'**entropies généralisées** ; lecture algébrique du paramètre d'ordre des transitions de
phase entre entanglement wedges ; poids à valeurs opérateur. ⟹ « Two Areas » = **deux surfaces
extrémales**, arène **AdS** — **ni** « deux modes ajoutés interchangeables », **ni** de Sitter.

**C-3 — FAITHFULNESS (S3).** Le cadrage libelle `2511.00622` : « modes de time-shift = horloge ».
Le résumé **ne pose pas cela**. Il pose : nécessité d'imposer **toutes** les contraintes du groupe
d'isométrie `SO(1,d)` (instabilité de linéarisation) et non le seul sous-groupe `R × SO(d−1)` ; les
isométries génériques de dS ne préservant pas un patch statique fixe, les contraintes ne peuvent être
implémentées dans une algèbre locale fixe ; **observateur covariant** dont la géodésique se transforme
covariamment ; à la quantification, superposition de géodésiques, patchs statiques **fluctuants**,
cadre de référence quantique `L²(SO(1,d))` ; modèle d'action où une particule porte des **charges
conservées**, une par générateur d'isométrie dS ; algèbre invariante de jauge en **produit croisé
modulaire moyenné** sur patchs statiques et configurations de géodésiques. Les auteurs se présentent
comme **généralisation mathématique** de S5 (CLPW).
