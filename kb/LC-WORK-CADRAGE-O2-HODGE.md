## 1. La question (c) `[résidu de (b) relocalisé — NE PAS re-poser (b) ni (b′)]`

Acquis amont (scellés, **non re-dérivés**) :
- (b) (`LC-D-O2-P1` §2) : l'inversion de Penrose **swappe** source↔VEV (`g₀↔g₃`) ⟹
  `P_swap=[[0,1],[1,0]]`, **`s=+1`** (sign-neutre) ; l'involution conforme **nue** ne fournit **pas**
  le `−1`.
- (`CT-DUAL` S2) : `S²=−𝟙` (vp `±i`, BD=`+i`) ; le **`−1` de `S` est le carré du Hodge** (dualité
  E-M / gravitationnelle, `i^{d-1}` dS = **source unique du signe**).
- Donc la S complète se **factorise** : `S = Swap ∘ Hodge`. Le swap est réalisé par la jonction
  (établi (b)). **Le facteur manquant est le Hodge.**

**(c) :** **la jonction Dirichlet→Neumann (réciprocité `Ω·ω=−1`) réalise-t-elle aussi le facteur
Hodge — donc la `S` COMPLÈTE — ou seulement le swap sign-neutre, le Hodge vivant dans un secteur que
la jonction ne touche pas ?**

Si **la jonction transporte le Hodge** : `J = Swap ∘ Hodge = S`, **gate P1 franchie** (structurel ;
P2 restante).
Si **la jonction = swap seul** : `J = Swap ⊗ 𝟙_Hodge ≠ S`, le `−1` est dans un secteur **non réalisé**
par la jonction, **(C-O2) NON établie** par cette voie, O₂ reste `à inventer`.

## 2. L'objet à figer — le fait structurel `[gelé AVANT fetch/algèbre]`

- **Le fait à trancher** : axe-A et axe-H **coïncident-ils** (la jonction se factorise `J=Swap∘Hodge=S`)

  non transporté) ?

  des axes, équivalent : réciprocité rotative E↔B vs diagonale), **pas** un signe extrait d'une parité
  (l'artefact de (b′)).

  `Ω` induit-elle une rotation de dualité sur le tenseur ?), et le **coefficient** O(1) éventuel

## 6. Prior honnête `[§6.4 — RÉVISÉ après réfutation de (b′)]`

- **Issue INDÉCISE, à fort enjeu — penchant DISCORDANCE renforcé par l'audit de (b′).**
  - **(i) Penchant discordance (renforcé).** Le diagnostic d'audit de (b′) a **lui-même** distingué
    deux opérations sur **deux axes** : le swap `Δ↦d−Δ` (axe-A radial, sign-neutre) et le `−1` issu de
    `Δ↦−Δ` / Hodge (secteur E-M). Si ces axes sont **indépendants**, la jonction (qui réalise le swap)
    **ne transporte pas** le Hodge ⟹ `J≠S`, (C-O2) non établie. C'est l'issue **par défaut** suggérée
    par la structure connue.
  - **(ii) Indice concordance, NON exploité.** `F6-MEMOIRE` place le secteur **TT-électrique** en
    **co-secteur de `g₃`** (le VEV, donc l'axe-A). Si l'axe électrique/magnétique (axe-H) n'est **pas**
    orthogonal à l'axe radial mais **lié** via `g₃`, alors la jonction **pourrait** transporter une
    composante du Hodge. **Cet indice n'a pas été testé** et **pourrait** rapprocher les axes — sans
    rien garantir.
  - **Tension réelle.** Le penchant (i) lit deux opérations comme **séparées par construction** ;
    l'indice (ii) suggère un **canal** (g₃ ↔ TT-électrique) qui les relierait peut-être. C'est
    **exactement** le hiatus radial↔tensoriel que (c)-G1/G2 doivent lever. **Le test doit pouvoir dire
    les deux.**
- **Garde d'honnêteté (post-(b′)).** Risque **accru** de **fit** vers l'issue agréable (`J=S`
  rouvrirait O₂) — c'est précisément ce qu'a produit (b′). Le firewall conditionnel (§4), le critère
  de verdict (§3) et l'**audit froid obligatoire au positif** (§5.5) garantissent que le transport de
  Hodge est **dérivé**, pas choisi. Ne **jamais** prélever un signe d'une opération qui ne réalise pas
  l'échange revendiqué : (c) demande **quelle** opération réalise le Hodge **avant** de lire son signe.
- **Même si `J=S` : `{A4 ; A2★ ; N}` INCHANGÉ.** Concordance = « gate P1 franchie », **pas**
