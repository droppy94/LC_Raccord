# Verdict de consolidation — VERROU-D1, jambe AMPLITUDE : `A_T=16/N` scellé ⟹ amplitude absorbée dans `N`

> **SANS sceau** (consolidation ; le sceau `A_T·N=16` préexiste, `LC-D-CT-ATN`, rejoué **EXIT 0**).
> **Constat de scoutage (anti-gabarit)** : le dernier front de D1 était **déjà tranché**. Complète la
> cartographie du verrou D1 (jambe amplitude ⟂ jambe sélection). Gel SHA in-fichier recouvrable (R-36).

## 0. Le constat (scoutage)

V25 listait « amplitude `A_T~1/N` » comme **prochain front OUVERT**. Le scoutage le révèle **déjà scellé** :
`A_T·N=16` (nombre pur, deux routes convergentes [A] canonique + [D] dS/CFT) et `C_T/N=1/(32π²)`
(`LC-D-CT-ATN`, `verif_D_CT_ATN.py` **EXIT 0**, rejoué). **3ᵉ gabarit pré-scellé de la session.**

## 1. La chaîne consolidée `[pièces scellées/gravées — aucune algèbre neuve]`

| pièce | rôle | statut |
|---|---|---|
| `⟨g₃⟩=0` | un-point | scellé (passerelle A3⟷D1) |
| `⟨g₃g₃⟩~k³` | forme deux-point | scellé (SPECTRE-K3 ; **non-perturbatif** via NONLIN-2PT) |
| `g₃=⟨T⟩` | dictionnaire | `g₀`=jauge, `g₃`=VEV (HOLOGRAPHIE-G3) |
| `résidu(D1)|2-pt = {A_T}` | recombinaison | un seul nombre (LC-WORK-D1-E-AMPLITUDE) |
| **`A_T=16/N`** | **rattachement** | **SCELLÉ** (`LC-D-CT-ATN`) |

## 2. Conséquence — l'amplitude est absorbée dans `N`

- L'amplitude **n'est PAS une hypothèse indépendante** : elle est **absorbée dans `N`** (`A_T=16/N`,
  scellé) — une **réduction de la liberté de D1**.
- **MAIS `N` n'est pas fixé** : `N≡Λ` (terminus, gravité quantique) ; circularité `LC-E` (`N=S_dS`
  présuppose `ℓ_P`) **non brisée** ; valeur/réalité de `C_T` en dS et survie au crossover = `à inventer`.
- ⟹ le résidu d'amplitude **hérite de l'irréductibilité de `N`** (`LC-D-IRREDUCTIBILITE-MOYENS` : `N≡Λ ← QG`).

## 3. Caractérisation COMPLÈTE du verrou D1

Avec le verdict d'axe (`LC-D-D1-VERROU-DELIMITATION` + amendement (ii)-levée) :

| jambe | verdict | nature du résidu |
|---|---|---|
| **sélection** (prescription) | **délimitation durcie** : aucun sélecteur **interne** | intrant **externe** requis |
| **amplitude** (`A_T`) | **absorbée dans `N`** (scellé `A_T=16/N`) | rattachée au **terminus `N≡Λ`** |

⟹ **D1 est ENTIÈREMENT CARTOGRAPHIÉ et NON fermable par les moyens INTERNES** : chacun de ses deux
résidus est soit **délimité** (sélection) soit **rattaché à un terminus** (amplitude). **Converge
décisivement** avec `LC-D-IRREDUCTIBILITE-MOYENS`.

## 4. Sans surclassement (`§6.4`) et périmètre

- Rattacher l'amplitude à `N` **réduit la liberté de D1** (l'amplitude n'est pas indépendante — résultat
  **déjà scellé**), **mais NE RÉDUIT PAS** le cœur `{A4 ; A2★ ; N}` : `N` demeure ; l'amplitude est
  **absorbée DANS `N`**, pas éliminée.
- **`{A4 ; A2★ ; N}` INCHANGÉ** ; `D1` non clos ; `N` non fixé (`≡ Λ`) ; `A4` non réduit ; `A2★` non
  tranché ; **CCC non démontrée NI réfutée**.
