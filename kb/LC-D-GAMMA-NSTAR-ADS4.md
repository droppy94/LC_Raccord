# Verdict de case — AXE γ, Phase 1, case G-3 : étalon-témoin AdS₄ de `𝒩★`

> **Zone PORTEUSE, scellé.** Re-dérivation de `𝒩★` **dans les axiomes propres d'AdS₄** contre le scoping
> gelé `cfaf5a2c…017f6e4`, sous l'amendement daté `e66e2a77…b9919d4`. Cadrage γ `a99f0e1b…c763d34`
> **intouché**. Sceau `verif_gamma_nstar_ads4.py` EXIT 0 (sha `367d2ddb…730444f`).
> **VERDICT = `NA-✓` (étalon établi)** ; **`établi (algèbre)` SOUS AXIOMES AdS₄ uniquement**.

## 0. Source témoin (R-41 confirmé)

de Haro `0808.2054`, « Dual Gravitons in AdS₄/CFT₃ and the Holographic Cotton Tensor ». Pièces lues :
éq.(1) `⟨T_ij⟩=(3ℓ²/16πG_N)g_(3)` ; éq.(14) développement FG `h=h₍₀₎+r²h₍₂₎+r³h₍₃₎` (pas de log,
terme linéaire absent) ; éq.(17) EOM TT `h̄''−(2/r)h̄'+□h̄=0` ; éq.(19)/(20) solutions `f₁,f₃` ;
éq.(23) régularité euclidienne ; éq.(24) oscillation Lorentzienne ; éq.(104) `⟨T⟩=(3/2)(ℓ/κ)²g₍₃₎` ;
**caveat p.3** ; **Ishibashi-Wald** invoqué p.7 (structure symplectique avec contre-termes) = l'outil
même de `𝒩★`.

## 1. Verdict par sous-condition (re-dérivé, non importé)

- **NA-1 — exposants indiciels.** Polynôme indiciel `s(s−d)`, `d=3` (graviton sans masse,
  `Δ(Δ−d)=m²ℓ²=0`) ⟹ `{Δ₋,Δ₊}={0,3}`. Recoupé par les solutions explicites : `f₁→1+O(r²)`
  (**source**, fall-off **lent**, branche `r⁰`) et `f₃→(p³/3)r³+O(r⁵)` (**VEV**, fall-off **rapide**,
  branche `r³=h₍₃₎`), conforme éq.(20).
- **NA-2 (PORTANTE) — normalisabilité.** Critère `Δ>d/2=3/2` : le mode **sous-dominant** `r³` (VEV) est
  **NORMALISABLE sans régulateur** (`3>3/2`) ; le mode `r⁰` (source) ne l'est pas (`0<3/2`). ⟹
  quantification **Dirichlet standard** bien définie, **l'objet renormalisé EXISTE** = valeur de
  **référence `𝒩★^AdS` « saine »** (la prémisse de G-3 est vérifiée).
- **NA-3 — control ≠ transport.** Le bord conforme AdS₄ est **timelike** : la finitude y tient grâce au
  bord timelike + la quantification standard. C'est le **CONTROL** de l'expérience, **pas** un transport.
- **NA-4 — caveat d'admissibilité (re-dérivée).** Le datum **alterné** Neumann/mixte (= la donnée
  structurellement pertinente pour la **jonction inter-éon / le transport**) est admissible ⟺ les **deux**
  modes normalisables ⟺ **fenêtre BF** `m²ℓ²∈(−9/4,−5/4)` (`d=3`). Le graviton `m²=0` est **HORS**
  fenêtre (`0>−5/4`) ⟹ admissible **seulement avec cutoff**. C'est **la face AdS du contournement**.

## 2. Firewall `mγ` (mordant — 3/3 au sceau)

`mγ-1` anti-blanchiment : branche VEV = `r³` par éq.(20), **PAS** importée de la lecture dS
`g₃=−(i/3)k³g₀` (SPECTRE-K3) — mutation `r²` rattrapée. `mγ-2` différence AdS-timelike↔dS-spacelike
**notée** (régularité euclidienne réelle éq.23 vs oscillation Lorentzienne éq.24). `mγ-3` **aucune**
continuation `ℓ→iℓ` (AdS pur `Λ<0`). `mγ-4` exposants nets `{0,3}` ⟹ `𝒩★` **bien-défini** (panel
intact ; mutation `d=0` dégénérante rattrapée). `mγ-5` anti-surclassement transfrontalier.

## 3. Finding `F-γ-1` (remonté — R-7, non silencieusement résolu)

Le label gelé **« mode lent (sous-dominant) »** est **ambigu** : « lent » collisionne avec le
**slow fall-off** de de Haro = `r⁰` (source, **non**-normalisable), alors que « sous-dominant » près du
bord = `r³` (VEV, **normalisable**) — **modes opposés**. L'intention du cadrage (objet = mode
réponse/sous-dominant, analogue dS de `g₃`) pointe `r³`, mais la collision crée un **risque `mγ-1`**
inter-cadres (re-dérivation future saisissant le mauvais mode). **Recommandation** : amendement R-7 daté
figeant **« `𝒩★` = normalisabilité du mode sous-dominant `r^{Δ₊}` (VEV / fast fall-off) »** AVANT le
premier transport (Phase 2). *Le présent verdict n'amende rien* (cibles γ + scoping intouchés) ; il
**consigne** le finding.

## 4. Lecture pour la matrice (en creux, NON un verdict de communauté)

L'étalon dit : en AdS₄ (bord **timelike**), le mode réponse `r³` est **normalisable sans régulateur**
(objet sain), et le datum **alterné** — celui qui compte pour la jonction — est **gaté par la fenêtre BF**
que le graviton **rate**, donc **cutoff requis**. La question du transport (Phase 2, dS spacelike-`𝓘⁺`)
devient nette : *le passage à `𝓘⁺` spacelike fait-il PERDRE la normalisabilité que le bord timelike
préserve ?* L'étalon fixe le **zéro de l'échelle** de « perte au transport ». **Aucune** convergence n'est
ici affirmée (G-4 exige `≥2` cadres indépendants re-dérivés).

## 5. Sans surclassement (§6.4 — transfrontalier) et périmètre

- **`établi (algèbre)` sous AdS₄ uniquement** ; aucun statut sous un autre cadre sans re-gel/re-dérivation.
- L'étalon est le **CONTROL**, **pas** un verdict de mur commun ni une réfutation de `H0`.
- **`{A4 ; A2★ ; N}` INCHANGÉ** ; `N` non fixé (`≡ Λ`) ; `A4` non réduit ; `A2★` non tranché ;
  **CCC non démontrée**.
